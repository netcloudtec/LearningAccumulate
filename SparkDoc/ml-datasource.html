Data sources (数据源)

在本节中，我们将介绍如何在ML中使用数据源来加载数据。除了一些常规数据源，如Parquet，CSV，JSON和JDBC，我们还为ML提供了一些特定的数据源。

目录

  • 图像数据源

Image data source (图像数据源)

该图像数据源用于从目录加载图像文件，它可以通过Java库中的ImageIO将压缩图像（jpeg，png等）加载到原始图像表示中。 加载的DataFrame 有一个StructType列: "image", 包含图像数据元信息的 image 结构体。该image 列的schema是:

  • origin: StringType (表示图像的文件路径)
  • height: IntegerType (图像的高度)
  • width: IntegerType (图像的宽度)
  • nChannels: IntegerType (图像channels的数量)
  • mode: IntegerType (OpenCV-compatible 类型)
  • data: BinaryType (在 OpenCV-compatible 有序的图像字节: 大多数情况下为row-wise BGR)

ImageDataSource 实现Spark SQL数据源API，用于将图像数据加载为DataFrame。

ImageDataSource implements Spark SQL data source API for loading image data as DataFrame.

In PySpark we provide Spark SQL data source API for loading image data as DataFrame.

In SparkR we provide Spark SQL data source API for loading image data as DataFrame.
